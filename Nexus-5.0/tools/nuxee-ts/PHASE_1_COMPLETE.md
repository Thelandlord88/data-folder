## 🌟 Usage Examples

### TypeScript API
```typescript
import { NUXEEEngine, PATTERN_LIBRARY } from '@nexus/nuxee';

const nuxee = new NUXEEEngine();
PATTERN_LIBRARY.forEach(p => nuxee.registerPattern(p));

const result = await nuxee.enhance(html, {
  auto_select: true,
  require_aaa: true
});

console.log(`Enhanced ${result.elements_enhanced} elements`);
```

### Command Line
```bash
# Enhance a file
nuxee enhance index.html

# List patterns
nuxee list-patterns

# Show stats
nuxee stats
```

### Conversational
```typescript
const response = await nuxee.ask({
  query: "Make my buttons more engaging"
});

response.recommended_patterns.forEach(p => {
  console.log(`${p.pattern.name}: ${p.relevance_score}`);
});
```

### Proactive
```typescript
const recommendations = await nuxee.analyzeAndRecommend(html);

recommendations.forEach(rec => {
  console.log(`${rec.priority}: ${rec.message}`);
  console.log(`Impact: ${rec.expected_impact}`);
});
```
